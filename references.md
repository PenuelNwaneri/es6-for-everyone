## By Wes Bos

# Module #1
1. `var` keyword 
    - Function scoped
2. `let` keyword 
    - Block scoped
    - Made to be updated
    - Declared once per block
    - Cannot access variable before it is defined
3. `const` keyword 
    - Block scoped
    - Made to be static
    - Declared once per file
    - Can mutate properties of the object
    - Cannot access variable before it is defined
4. `Object.freeze()`
    - Locks properties of an object
